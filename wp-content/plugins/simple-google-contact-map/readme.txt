Simple embedded Google Map to any Post or Page Using shortcode and Widget.

== Description ==
Now a Days Google Map is a must for all websites. I have created a simple google contact map plugin with nice features. Add your map using simple Shortcode and Widget. simple but powerful.

= Features Included =
* Simple to use with just add a shortcode in page or post.
* Drag a widget in your sidebar.
* List of map options in admin settings like (maptype, zoom level, mapwidth, mapheight etc)

= Usage =
* Once install the plugin go to settings menu -> Google Map-SGCM
* Fill the settings form and Save it.
* Add the following shord code to your page or post `[sgcm_map src='sgcm']`.
* Go to Appearance menu -> Widgets, Find `Google Map Widget-SGCM` and Drag to your sidebar.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Once install the plugin go to settings menu -> Google Map-SGCM
4. Fill the settings Form and Save it.
5. Add the following shord code to your page or post `[sgcm_map src='sgcm']`.
6. Go to Appearance -> Widgets, Find `Google Map Widget-SGCM` and Drag to your sidebar.
